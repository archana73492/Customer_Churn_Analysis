# Customer_Churn_Analysis


## 📌 Overview
This project predicts whether a customer will churn or not using machine learning.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib

## 📊 Model Performance
- Accuracy: 83%
- Confusion Matrix:
  [[1407 195]
   [154 244]]

## 📈 Evaluation Metrics
- Precision
- Recall
- F1-score

## 📷 Visualization
Includes:
- Count plot (Churn vs Not Churn)
- Confusion matrix heatmap

## 🚀 Steps Performed
1. Data Preprocessing
2. Train-Test Split
3. Model Training
4. Prediction
5. Evaluation

## 📌 Conclusion
The model performs well with good accuracy, but improvement is needed for churn class prediction.
